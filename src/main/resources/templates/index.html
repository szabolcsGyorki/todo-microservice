Todo API microservice for codecool.

Usage:
Retrieve the list of todos from the given user:
https://codecool-todo-microservice.herokuapp.com/todo/{name}
GET https://codecool-todo-microservice.herokuapp.com/todo/jeff

Add new todos for the given user:
https://codecool-todo-microservice.herokuapp.com/todo/{name}/{title}
GET https://codecool-todo-microservice.herokuapp.com/todo/jeff/todo

Complete the todo:
https://codecool-todo-microservice.herokuapp.com/todo/{name}/{id}/complete
GET https://codecool-todo-microservice.herokuapp.com/todo/jeff/2/complete

Example response:
[ { "id":1, "title":"first todo", "completed":true }, { "id":2, "title":"another one", "completed":true } ]
